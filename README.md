# 🚀 Daily Move Routine

Esta é uma landing page responsiva focada em bem-estar e saúde, desenvolvida para praticar conceitos fundamentais de Front-end.

## 📋 Sobre o Projeto 
O projeto apresenta uma rotina matinal que pode ser feita de qualquer lugar, seja em casa, no parque ao ar livre, etc. Eu escolhi esse tema porque gosto de temas de saúde, praticidade. Foi um ótimo exercício para consolidar conhecimentos de estruturação e estilização. 

##  🛠️ Tecnologias utilizadas
* **HTML5:** Estruturação semântica.
* **CSS3:** Estilização e efeitos de Hover.
* **Flexbox:** Usei para alinhar os cards e deixar o layout organizado.

## 🧠 O que eu aprendi 
Neste projeto, eu foquei em:
* Organizar as classes do CSS de um jeito mais claro.
* Praticar o fluxo do Git (add, commit e push) que eu não fazia já tinha um tempinho e estava enferrujada! 😅
